# Service Specials Google Sheet Structure

## Sheet Name: "Service Specials Automation"

### Required Columns:

| Column Name | Description | Example |
|------------|-------------|---------|
| Dealership | Exact dealership name (must match workspace config) | Swickard Honda Thousand Oaks |
| Hero_Image_URL | URL to the hero banner image | https://enfoilk.stripocdn.email/.../hondato_1sl.jpg |
| Schedule_Service_URL | Link to schedule service page | https://www.hondaofthousandoaks.com/schedule-service.html |
| Service_Specials_URL | Link to service specials page | https://www.hondaofthousandoaks.com/service-parts-specials.html |
| Service_Match_URL | Link to service match page | https://www.hondaofthousandoaks.com/service-match.html |
| Phone_Number | Display phone number | 805-371-5500 |
| Phone_Number_Tel | Phone number for tel: link | +18053715800 |
| Brand_Color | Hex color code for brand accent | #00adef |
| Dealership_Display_Name | Full dealership name for disclaimers | Swickard Honda Thousand Oaks |
| Offer_Expiration_Date | Date the offer expires | 10/31/25 |
| Service_Special_Disclaimer | Full disclaimer text for service specials | Plus tax and applicable fees... |
| Service_Match_Disclaimer | Full disclaimer text for service match | Dealership must be within 25 miles... |
| Brand_Make | Vehicle brand (Honda, Toyota, etc.) | Honda |
